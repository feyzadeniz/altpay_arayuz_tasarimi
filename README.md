# AltPay Arayüz Tasarımı
Bu çalışma staj sürecimde takım halinde geliştirdiğimiz AltPay Ödeme Sistemi projesinin arayüz tasarımıdır. İş analizi ve akışları da kendim tasarladım.

Projem web sayfası olarak geliştirildi. Hem ürünün tanıtımı için hem de henüz hiç kullanmamış müşterilerin kayıt olup başvuru yapmaları için çeşitli yönlendirmeler içeriyor. Başvuru onaylandıktan sonra ise müşteri, kullanıcı panelini web üzerinde kullanabiliyor.

Ürünün kullanım amacı ise kullanıcılarımızın kendi müşterilerinden dijital şekilde, kolayca ödeme alabilmesi. Kullanıcılarımız link oluşturabiliyor, bunları müşterilerine ileterek ödeme alıyor. Ödeme durumlarını kolayca takip edebiliyor ve işleyişin özetine ulaşabiliyor.

Son kısımda ise akışın müşteri tarafını görüyor olacaksınız. Linke erişen müşteri, oluşturduğumuz arayüzde gerekli bilgileri girerek ödeme işlemini gerçekleştiriyor.

Çalışmamı incelediğiniz için teşekkür ederim :)
